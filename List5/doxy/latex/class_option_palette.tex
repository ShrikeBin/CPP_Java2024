\doxysubsection{Detailed Description}
A palette for organizing Option\+Buttons in a grid layout. 

This class represents a palette for organizing Option\+Buttons in a grid layout. Option\+Buttons can be added to the palette, and they will be automatically positioned in a grid based on the order of insertion.
